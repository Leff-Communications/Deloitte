Doctors on Call

Program description

Deloitte's Doctors on Call program is a paid benefit available for all USI professionals and their dependents. This program provides access to health risk assessments, a health portal (to track fitness, etc.), and unlimited medical tele-consultations with specialist doctors. The program also offers various value-added services including discounted medicine delivery, nursing support, health check-up, and more.

Resource information

  • If you need more information or want to use the benefit, please access Doctors on Call

  • You can also write to the USI ELE Benefits team: usindiaotherbenefits@deloitte.com

Bright Beginnings

Program description

Bright Beginnings is Deloitte’s paid maternity management program for all USI women professionals and spouses of men professionals. This program provides access to a dedicated program coordinator who helps schedule expert consultation calls with medical specialists including gynecologists, physiotherapists, psychologists, and dietitians.

Resource information

Live Well Program

Program description

Live Well is our Employee Assistance Program (EAP) that helps professionals and their families with many of their well-being needs.

This program offers counseling services provided in complete confidentiality, whether you are dealing with stress, life challenges, or need mental health assistance. It also offers advice on how to make healthy lifestyle changes.
